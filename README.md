# Product Teardowns

PM-style teardowns of products I actually use — with an evaluation mindset.

My angle: I come from psychology + measurement (UPenn). So beyond "what feels off," I sample real usage, label failures, and trace problems back to the metrics a product is (implicitly) optimizing. Less vibes, more evidence.
